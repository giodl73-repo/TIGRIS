# Codenames — Concept (review-import)

## Anchor declaration

**Anchor persona:** Chvátil
**Anchor axis:** B5 Architectural Novelty (adopted; monitoring at 2.0R)
**Anchor claim:** "Codenames' information-architecture is genuinely novel: a Spymaster communicates via a single word clue + integer count, encoding semantic relationships across a shared 5×5 word grid that both teams see but interpret differently. The mechanism — one-word clue spanning multiple semantically-adjacent targets while avoiding enemy words and the Assassin — has no prior art in party/word games. Knizia's Ra uses bidding-information architecture; Codenames uses semantic-overlap-information architecture. Structurally distinct. Chvátil own-anchor close-out (last own-anchor TtA #8, 17 games ago)."

**Dormancy-watch note:** C2 Minimum-Score Shape is in dormancy-watch (v2.22). Codenames has binary round win/lose — this is almost certainly a C2 retire-explicit. The targeted earn window for C2 (games #25, #26, #27) opens here; Codenames is game #1 of the 3-game window and is very unlikely to earn C2. This is acceptable — the window allows C2 one more pass at Nest (#26) and Furrow (#27) before de-adoption review fires.
