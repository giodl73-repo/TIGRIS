# Orléans — Design (Rules Import)

## Core loop

18 rounds. Each round: draw follower tiles from bag → place tiles on action spaces (combos: 2–3 specific followers = 1 action) → take action → refill bag (discarded tiles go back) → Event Phase.

## Actions

- **Farm:** gain grain resource.
- **University/Scriptorium:** gain scholars; advance on tech tracks.
- **Tradepost:** place merchant on road/river network; open trading posts.
- **Build:** place buildings for income and bonus tiles.
- **Monastery:** gain monks (versatile followers).
- **City:** reach cities for bonus rewards.
- **Place Followers:** add new follower tiles to bag.

## Followers (tiles)

Knight, Farmer, Scholar, Boatman, Merchant, Monk, Builder. Citizens (special, bought). Each action requires a specific combination from 2–3 follower types. E.g., Trade requires Merchant + Boatman + one other.

## Event Phase

Each round a random Event tile fires: Plague (lose followers from bag), Taxes (lose coins), etc.

## Scoring

Progress tracks: Farming, Shipping, Scholars, Commerce, Cathedral (each has 8 levels; higher levels score more). Trading posts = 1 VP each. City tiles = varying VP. Coins (1 VP per 5). Citizens (special scoring). Board presence on 48-node river network.

## Weight profile

Weight 3.04. Teach time ~25 minutes. Bag-building is novel to Euro players but intuitive once the draw-to-action mapping is clear.
