# Terra Mystica — Design (Rules Import)

## Core loop

6 rounds. Each round: Income → Actions (in turn order, pass to end the round) → Cult Bonus → Cleanup. Actions include: terraform + build, upgrade dwelling → trading post → stronghold/sanctuary, send priest to cult track, build bridge, use power action, convert resources, advance on bonus round tile.

## Resources

Workers (for terraforming), Coins, Priests (for cult), Power (3-bowl cycling system), Spades (from actions/transformations).

## Terrain and expansion

7 terrain types (Plains, Swamp, Lakes, Forest, Mountains, Wasteland, Desert). Each faction owns 1 type. Building on terrain requires it to match faction type. Adjacent terraforming costs workers × steps between terrain types (0–3 steps on a cycle).

## Factions (14)

Halflings (cheap terraforming), Witches (flying start), Darklings (power magic), Auren (favor tiles), Swarmlings (fast town), Engineers (bridges), Dwarves (underground tunneling), Giants, Nomads, Fakirs (sandstorm), Mermaids, Chaos Magicians, Alchemists, Cultists. Each has unique power ability, starting layout, special buildings.

## Scoring

- Round-end scoring tiles (change each round): N shipping / N terraforming / N dwellings built, etc.
- Town founding (3–4 structures adjacent): score town tile.
- Final scoring: Merchant building count, Fortress/Sanctuary count, Largest connected area, Cult track positions (fire/water/earth/air), Residual power/resources.
- Total: typically 100–170 VP for winner.

## Key tension

Resource triage: workers → terraforming space or upgrade buildings? Priests → cult track or town founding? Power → gain resources or convert to coins? Every resource has 3+ uses; shortfall is constant.
