# Kramer-Kiesling review — ZEN PATH concept

*"We want the game a nine-year-old can play and her father can't beat her at without thinking."*

ZEN PATH has the bones of a family-to-expert design — one action per turn, public information, aspect-gated jumps. The premise cross-breeds Knizia's *Samurai* minimum-score shape with a track-race geometry. We have built our careers on exactly this sort of architecture, so we will evaluate with the discipline of our own shelf: *Tikal*, *Torres*, *Mexica*, and — the one with the tournament scene — *Azul*.

## C7 Action-Menu Clarity — **8.0** (3p), **8.0** (4p)

The one-action-per-turn structure is tight. Draft a tile, play its effect, resolve. That is cleaner than *Tikal*'s 10-point economy at the cost of strategic texture, and cleaner than *Mexica*'s 6-point menu at the cost of pacing control. The seven tile types (Step / Ascend / Focus / Ladder / Chute / Duel / Meditate) sit at the upper edge of the C7 anchor (menu size ≤ 6) — one over, defensible if iconography does the work. We would push the designer to collapse Focus into a stronger Ascend variant (same icon, marked value) to get menu size to six and save a rules-sheet paragraph. The Meditate tile at 2-of-55 earns its oddity only if the teach mentions it once and the icon is self-explanatory.

## D1 Family-to-Expert Scaling — **7.5** (3p), **7.0** (4p)

The scoring equation — **position × MIN(Honor, Strength, Strategy)** — is where family-to-expert either earns its scaling or fails. For a ten-year-old, the multiplication step is grade-school arithmetic; the MIN function is the harder half. *Azul* sidesteps this by scoring in row/column units a child can count with a finger. *Samurai* gets away with it because aspects are colored majorities on a map, not numbers. ZEN PATH asks the child to compute MIN across three tracks and multiply a number up to 40 by a number up to 10. That is manageable, but it is not *Azul*-simple. Mitigation: during play, each aspect track should display the player's current score as position × min at all times — a running total printed on the board. Samurai's partial-capital scoring in *Tikal* works because the total is always visible; do the same here.

Tournament depth: the aspect-gated jumps and player-placed chutes give the veteran something to chew on. We score this adopted axis a respectable 7.5, not a 9, because the scoring arithmetic is a harder teach than *Azul*'s and a harder board-read than *Samurai*'s.

## D2 Spatial-Interaction Presence — **4.5** (3p), **5.0** (4p)

This is where we push back hardest. A 40-square linear path is not spatial in the sense that *Tikal*'s pyramid or *Mexica*'s canal-blocking is spatial. The path is a track, and a track has one dimension. Duel tiles and player-placed chutes provide adjacency pressure, but adjacency on a line is poverty compared to adjacency on a plane. The printed path *could* become spatial if the zigzag geometry allowed diagonal threats or branching, but the concept as written reads as linear with jumps. The D2 axis anchor is "chess, Mexica canal-blocking, Tikal pyramid competition." ZEN PATH scores below 5 here. If the designer wants D2 to earn, the path needs branching forks where choosing the left trail vs the right trail creates spatial competition — not just temporal racing. Otherwise this axis is a predicted refute, correctly.

## A6 Teachability — **7.5** (3p), **7.0** (4p)

The stated teach target is 15 minutes. For a 35-45 minute game at weight 2.5, that is the correct ceiling (our *Azul* teach is 8 minutes; our *Tikal* teach is 15 minutes for a 90-minute game). ZEN PATH sits in the middle. Achievable only if: (1) the scoring equation is a single printed card shown during teach, (2) the seven tile types reduce to six with collapsed iconography, (3) pre-printed path features use the same icon grammar as player-placed features so "a ladder is a ladder." If the designer hits all three, 15 minutes holds. If any slips, the teach bleeds to 22 minutes and the family audience evaporates.

## Summary

ZEN PATH earns its C7 and A6 with discipline. D1 is adequate but not *Azul*-class. D2 is the weak link — a linear path cannot carry a spatial claim. Fix the path geometry or retire the D2 prediction.
