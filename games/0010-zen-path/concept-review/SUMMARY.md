## Key clusters (multi-voice concerns)

### Cluster 1 — Architecture is too comfortable (4 voices — HIGHEST priority)

- **Chvátil (B5 3)**: "comfortable, too comfortable" — Thebes-skin; no load-bearing weirdness.
- **Lacerda (B1 3)**: Lisboa test fails — remove player-placed ladders/chutes and the game still works. Subsystems ADD, not MULTIPLY.
- **K-K (D2 4.5)**: 40-square line is not spatial in the Tikal/Mexica sense; needs branching forks.
- **Rosenberg (C4 3)**: no durable engine-garden layer; tiles are context-free.

Concrete fix candidates:
- **A Path rotates between rounds** (Chvátil's Tzolk'in-proposal) — paths shift each round, ladders relocate
- **B Branching forks** (K-K) — path has 2-3 forks; players pick based on aspects
- **C Parallel tracks** — 3 parallel paths instead of 1
- **D Aspect-as-position** — aspects ARE positions on the path (radical; max gearing)

### Cluster 2 — Scoring dominance (Knizia + Feld, 2 voices)

Knizia's arithmetic: position 35 × MIN 5 = **175** beats balanced (20 × 5) = 100. Race-plus-one-aspect may dominate.

Feld's attack: `position × MIN` is a single curve, not incommensurable. C6 Point-Salad fails by construction.

Fix candidates:
- **A Partition Step tiles by aspect** (Knizia) — 6 Honor-Steps, 6 Strength-Steps, 6 Strategy-Steps. Movement IS aspect-commitment.
- **B Steeper min-score**: position × MIN² instead of × MIN.
- **C Both**.

### Cluster 3 — 2p/solo blocker (Stegmaier, strong)

Stegmaier D3 4/10: 2p-as-variant is hard-blocker; no solo is soft-blocker.

Fix: spec 2p as full mode in concept (not variant); add solo-mode sketch.

### Cluster 4 — Tile economy (Rosenberg, Vaccarino)

- Rosenberg: market too abundant; no feeding-equivalent
- Vaccarino: 18 Step tiles too many (12-14 better); bypass chain depth

Fix: reduce Step count; add periodic "samurai's debt" obligation.

### Cluster 5 — Info-transparency cost (Lacerda alone but structural)

Lacerda B4 2/10: zero-hidden-info surrenders the adopted axis entirely.

Fix candidates:
- **A Private hand instead of public market** — tiles go face-down to hand
- **B Some secret placements** — Chute tiles placed face-down
- **C Accept** — B4 scores low; proceed

### Cluster 6 — Small tile refinements (Vaccarino)

- Duel tile needs card-only resolution (no rulebook lookup)
- Meditate count 2/55 too thin; expand to 4-5

## Recommended revision path

**Adopt P1=B (branching forks), P2=A (partition Steps), P3=A (2p + solo sketch), P4=A (reduce + feed), P5=A (hand)**.

This is a bigger revision than UNFOLD's. Concept v1.1 would essentially re-architect the path (forks), re-partition tiles (aspect-specific Steps), and add a private-hand layer.

Alternative: **less revision** — apply only P1 + P2 (the high-priority clusters) and accept the remaining issues as design-stage tuning.
