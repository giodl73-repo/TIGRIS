# HEARTH — Concept

## Premise

HEARTH is a 1–5 player cooperative campaign game in which the campaign IS the learn-cycle. Game 1 ships with 6 rules and a 30-minute play. Each subsequent session unlocks exactly 1 new rule and 1 new role, delivered in sealed envelopes opened mid-session when a trigger condition is met. By session 8, players have the full expert ruleset — but they earned each rule through play rather than reading it upfront.

The design bet: **teach-by-doing is architecturally superior to teach-by-reading.** Standard games hand players a rulebook. HEARTH hands players 6 rules, lets them play, then reveals the 7th rule exactly when the game creates the moment where that rule is needed. The rule lands with context it immediately uses.

The legacy envelope IS the teach. After session 8, a player can teach HEARTH to a new group from session 1 — the unlock sequence is re-doable because the envelopes can be resealed (non-destructive legacy: stickers are not used; instead, cards are added to the ruleset from sealed envelopes that can be replaced).

## Players and length

- **Players:** 1–5
- **Length:** 30–60 min (session 1 = 30 min; session 8 = 60 min)
- **Weight:** 1.5 → 3.0 (session 1 is family-weight; session 8 is medium-Euro weight)

## Anchor mechanic

**Incremental Rule Revelation.** Session 1 contains: Move / Gather / Build / Rest / Threat / End-Trigger — 6 actions on a minimal board. Session 2 opens Envelope B: adds the Craft action + the Craftsperson role. Session 3 opens Envelope C: adds the Defend action + the Defender role. By session 8, all 8 actions and all 8 roles are in play.

The rule-unlock timing is non-arbitrary: each envelope triggers when the game creates a moment where the new rule would have been useful. "You've gathered 3 times this session without being able to do anything with surplus resources — open Envelope B." This ensures the new rule arrives with a felt need.

## Artifact-as-story

**The sealed Envelope stack** (8 envelopes, A–H). Each envelope contains: 1 new Action card (added to the table), 1 new Role card (added to the role pool), 1 rule insert (read aloud), and 1 Unlock Narrative card (2 sentences of flavor). The physical envelopes are the game's progression system. Non-destructive: all envelopes can be resealed; the unlock condition is printed on the outside so a second group can follow the same sequence.

## Inspiration / lineage

- **Stegmaier / Charterstone (2017)** — legacy campaign where each session unlocks new components and rules. HEARTH borrows the "campaign = learning arc" design ethos but removes permanent damage (no crossed-out rules; no stickers).
- **Leacock / Pandemic (2008)** — cooperative disease-management; accessible base with escalating complexity. HEARTH borrows the cooperative structure and threat-escalation tension.
- **Vaccarino / Dominion (2008)** — incremental complexity: new card types expand the rule space each expansion. HEARTH borrows the "rules added one at a time" expansion design.

HEARTH's architectural claim: **non-destructive legacy-as-teach.** Unlike Pandemic Legacy (destructive: stickers, torn cards) and Charterstone (semi-destructive: permanent stickers on board), HEARTH uses only additive envelopes that can be resealed. Any group can play through the full 8-session arc; then another group can start at session 1 with the same copy.

## Target review in the TIGRIS pipeline

**Earn-candidates:**
- **A6 Teachability (anchor; Stegmaier)** — HEARTH's entire architecture is a teachability delivery system. Session 1 is the most teachable cooperative game in the corpus (6 rules, 30 min). The unlock sequence is teachability-by-context: each new rule arrives when the game has already created the moment where that rule is needed.
- **D1 Family-to-Expert Scaling (K-K)** — Session 1 = family-weight; session 8 = medium-Euro weight. The scaling gradient is the entire design.
- **B2 Catastrophe Pressure (Feld)** — cooperative threat escalation across sessions; the Threat mechanism escalates if not managed.
- **A3 Interaction (Chvátil)** — fully cooperative; mandatory discussion at every role-action decision.
- **A7 Emergence-Replayability (Lacerda)** — non-destructive legacy: the campaign arc can be replayed with a fresh group. Each run produces different strategic patterns from the same unlock sequence. Distinctly different from Pandemic Legacy's zero-replayability.
- **C8 First-Turn Compression (Stegmaier secondary)** — session-opening setup changes each session as new components are added; each session begins with a different setup overhead.

**Collision candidate:**
- **A6↔D1** — are teachability (A6: can I teach this to new players?) and family-to-expert scaling (D1: does the game reward skill growth?) orthogonal or the same question? In HEARTH they may be the same question (the game is designed so that both are achieved by the same mechanism: incremental rule revelation). OP candidate: A6 = per-session entry point; D1 = cross-session progression arc.

**Retire-explicit candidates:** B5 (HEARTH acknowledges Charterstone/Pandemic Legacy lineage; no novel architecture claim), C6 (no incommensurable scoring), B3 (no conversion chain), D4 (lite on late-game lock-in — sessions are self-contained).

## Non-goals

- No permanent damage to components — HEARTH is explicitly non-destructive legacy.
- No scoring track — HEARTH is cooperative win/lose per session.
- No competitive mode.
- HEARTH does not claim B5 Architectural Novelty (non-destructive legacy envelope is an incremental improvement on Charterstone/Pandemic Legacy, not a new architecture).

## Open questions (to resolve during DESIGN)

1. **Trigger conditions for envelopes**: how specific should the trigger be? "After your third Gather without using surplus" is precise but may be gamed. "After session 2" is too blunt. Balance: thematic but deterministic.
2. **Role count**: 8 roles for up to 5 players means players hold multiple roles by session 8. Is role stacking the right solution?
3. **Non-destructive resealing**: does printing the unlock condition on the envelope exterior break the surprise for careful readers? Proposed solution: envelope exterior shows only a silhouette/icon; a separate Condition card (face-down in box) shows the text.
4. **2p vs 5p cooperation dynamics**: cooperative games scale differently. How does a 2-player HEARTH session compare to a 5-player session when only 2 roles are in play (session 2)?
