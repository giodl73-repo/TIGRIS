# Tier-A Stakes — HEARTH

## Anchor

**Stegmaier** anchors **A6 Teachability** (adopted). Claim: "HEARTH's architecture IS a teachability delivery system. Session 1 requires under 5 minutes to teach (6 rules; cooperative; 30 min play). Every subsequent rule arrives at the moment the game creates a felt need for it — the unlock trigger ensures no rule precedes its context. Teach-by-doing is the game's structural bet."

Adjacency partner: **K-K on D1** (different persona). A6↔D1 is the designed collision: session-1 entry teachability (A6) vs. cross-session progression arc (D1) — temporal-register OP candidate.

## Drafts (24)

Round 1: Knizia → Rosenberg → Feld → Lacerda → Chvátil → K-K → Vaccarino → Stegmaier.

### Knizia
1. **C1 Tension Budget.** 7/7/7. "Threat track (0–8; loss at 8) is the tension clock. Event deck adds ~0.5 Threat per round; mitigation costs actions. The tension budget is explicit: players count rounds to loss and plan mitigation timing. Earn."
2. **A5 Downtime/Pacing.** 5/5/5. "Cooperative discussion is the downtime source. 2 actions per player; 5 players × 2 actions = 10 decisions per round. Session discussion overhead. Hold."
3. **C2 Minimum-Score Shape.** 2/2/2. "Binary win/lose per session. No VP floor. Retire-explicit (C2 cleared; ordinary accumulation)."

### Rosenberg
1. **D4 Late-Game Lock-in.** 6/6/6. "Structure tiles placed in early sessions persist into late sessions. Session 3's Storage tile is still on the board in Session 8 — that placement is a permanent lock-in. Hold — present but moderate (cooperative, so lock-in is a collective benefit rather than a strategic commitment)."
2. **C3 Scarcity Bite.** 6/6/6. "Session 1: players start with 2 Food, 1 Wood; Build costs 2 Wood + 1 Stone. First action must be Gather. Resource scarcity is immediate and canonical. Earn."
3. **B3 Conversion Chain Depth.** 5/5/5. "From Session 2: Gather (resource) → Craft (2 resources → 1 good) → Build (good + resources → structure). Depth-3 pipeline. Hold — present but chains are cooperative, not per-player competition."

### Feld
1. **B2 Catastrophe Pressure.** 7/7/7. "Threat reaching 8 = cooperative loss. As sessions advance, more Event cards per deck add Threat faster; new Threat sources arrive in later envelopes. Multi-level escalating catastrophe. Earn."
2. **A2 Decision Density.** 6/6/6. "Per-action cooperative decision: which action serves the current shared goal (Gather for resources, Build for win condition, Rest for threat)? With multiple roles, the per-player decision space expands. Earn."
3. **C6 Point-Salad Incommensurability.** 1/1/1. "Cooperative win/lose; no scoring dimensions. Retire-explicit."

### Lacerda
1. **A7 Emergence-Replayability.** 8/8/8. "Non-destructive legacy: envelopes are resealable; a second group can play the full 8-session arc from Session 1. Unlike Pandemic Legacy (zero replayability), HEARTH's campaign is designed to be replayed. Additionally: each run produces different strategic patterns from the same unlock sequence (different player compositions, different event draws). A7 earns strongly. Earn."
2. **B1 System Gearing.** 6/6/6. "Resources → actions → structures → reduced threat → session win. 4-gear cooperative pipeline. Earn."
3. **B4 Information-Transparency-Cost.** 7/7/7. "Sealed envelopes are the most explicit information-cost design in the corpus outside Pandemic Legacy: future rules are physically inaccessible until triggered. Players cannot plan for Session 6 rules during Session 2. B4 earns — hidden future ruleset = real information cost. Earn."

D1's canonical earning condition: the game serves both entry-level and expert players via the same mechanism (unlock sequence). Earn."
