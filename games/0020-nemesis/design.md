# Nemesis — Rules Summary (Review Import)

Condensed summary for review purposes. Canonical rules in the 2018 Awaken Realms rulebook. Fair-use: critical commentary; no verbatim rulebook text beyond minimum needed to ground the Parliament stakes.

## 1. Overview

1-5 players wake from cryosleep on the disabled spacecraft *Nemesis*. The ship is infested with alien Intruders. Each player has a public character with unique abilities + TWO private Objective cards (usually 1 kept, 1 discarded at first opportunity). Win conditions differ per objective — some align with group survival, others require specific players to die or specific outcomes. The game is **semi-cooperative**: players can genuinely cooperate OR act on private agendas. The ship will self-destruct on a countdown; Intruders will evolve and mass; fires and contamination escalate; the Queen will arrive.

## 2. Components (summary)

- 15 Room tiles (randomly revealed as explored).
- 5 Character boards with unique abilities.
- Intruder bag with 10+ tokens: Larva (threat=0) / Adult (2) / Breeder (3) / Queen (5) + Blip tokens (unknown threat until revealed).
- Intruder Attack cards (for Combat resolution).
- Personal Objective cards (~30 unique; each player draws 2, keeps 1).
- Contamination cards (infect players via bite/melee).
- Event deck (per-round events including power failures).
- Noise markers + Encounter tokens.
- Action cards per character (hand size 5-7).

## 3-6. Rules sections (condensed)

### 3. Setup

Players choose characters; draw 2 Objectives (keep 1 secret, discard 1). Place Hibernatorium + 1 revealed room. Shuffle Intruder bag with starting mix. Place self-destruct timer + fire tokens. 1-hour to 3-hour game length estimate.

### 4. Turn structure

Each round has 5 phases:
1. **Event phase** — reveal event card; apply effects (typically: self-destruct advance, Intruder entry, power failures).
2. **Action phase** — each player takes 2 actions, simultaneously (hand-cards represent action options).
3. **Intruder phase** — revealed Intruders move and attack per their AI cards.
4. **Noise phase** — noise tokens in rooms trigger encounter checks (draw from Intruder bag; blip or Intruder revealed).
5. **Cleanup** — hand refresh, round counter advance.

### 5. Actions

~8 action types per character (Move / Shoot / Melee / Search / Trade / Interact-with-room / Stealth / Surveillance). Icon-coded on hand-cards. Each player has 2 actions per round.

### 6. Scoring / end condition

No points. Each player individually either **succeeds** (their private Objective's conditions met at game end) or **fails**. Common end-triggers:
- Ship destroyed (self-destruct fires).
- Queen arrives and slaughters everyone.
- All players escape via escape pods.
- Specific player or character dies/survives per objective.

Multiple winners possible (all players whose objectives succeed). Zero winners possible (total party kill + objectives unmet).

## 7. Edge cases & clarifications

- **Blip tokens**: noise-triggered encounters can reveal blank blips (no Intruder) or real Intruders. Fog-of-war.
- **Contamination**: bitten players draw Contamination cards (some kill immediately; some are time-bombs).
- **Objectives revealed at end**: players don't reveal objectives mid-game; strategic play is obscured.
- **Power rooms**: specific rooms control life-support, weapons, comms; disabling or capturing them is often an objective.
- **Character asymmetry**: Scientist / Soldier / Mechanic / Medic / Pilot / Captain / Intruder-Proximity-Scout (across expansions); base 5 characters each with unique ability + hand composition.
