# A Few Acres of Snow — Concept

## Anchor declaration

**Anchor persona:** Vaccarino
**Anchor axis:** C4 Engine-Garden Dependency (adopted)
**Anchor claim:** "Deck-building is the engine-garden mechanism. AFoS applies my signature mechanic outside Dominion's frame — deck + area-control fusion. Adopted-axis canonical re-earn expected. But: Halifax Hammer evidence threatens my own claim — if one deck shape dominates, the 'garden' is degenerate."

**Secondary focus:** Stegmaier C8 for 2nd earning → **Pool adoption target (Pool → 24/25 = 96%)**.

**Tertiary focus:** Lacerda A7 Emergence under **Halifax Hammer refute pressure** — candidate for **first refute of an adopted axis** across 16 games.

## Context

- Second consecutive non-Euro-but-non-Euro-style (2p deck-building + area control; Wallace is Euro-adjacent).
- Second non-own anchor (Wallace not in 8-persona roster).
- "Halifax Hammer" = optimal British opening settles Halifax first turn, spams London cards via deck manipulation, locks out French response. Widely considered solved at expert level.
- AFoS is a known failure-of-playtesting case study; Wallace himself acknowledged the exploit.

## Expected signals

**Earn-candidates:**
- C4 Engine-Garden (anchor; Vaccarino) — deck-building canonical.
- C8 First-Turn Compression (Stegmaier) — Halifax Hammer IS first-turn solved opening.
- B5 Architectural Novelty (Chvátil) — deck-building + area-control fusion novel in 2011.
- A3 Interaction (Chvátil) — 2p zero-sum direct conflict; ambush mechanic.
- D4 Late-Game Lock-in Point (Rosenberg) — fortifications lock late-game map.
- B1 System Gearing (Lacerda) — deck ↔ map ↔ resources ↔ military ↔ scoring.
- B4 Information-Transparency-Cost (Lacerda) — hand cards + hidden ambushes.
- D2 Spatial-Interaction (K-K) — actual geographic map with adjacency.
- A2 Decision Density (Feld) — card-dual-use (location vs action vs asset).
- A4 Variance Calibration (Vaccarino) — draw variance with mitigation via siege/reserve.

**Retire-candidates:**
- C6 Point-Salad (Feld) — single VP track.
- D1 Family-to-Expert (K-K) — 2p-only, weight ~3.2.
- D3 Count-Robustness (Stegmaier) — 2p-only hard cap.
- B6 Scoring Multiplier (Chvátil) — additive VP.
- A6 Teachability (Stegmaier) — moderate-low (not as heavy as TS).
- B2 Catastrophe Pressure (Feld) — few catastrophic events.

**Refute-candidate:**
- **A7 Emergence-Replayability (Lacerda)** — **Halifax Hammer as structural refute.** Solved-game evidence. Would be first refute of an adopted axis across 16 games.
