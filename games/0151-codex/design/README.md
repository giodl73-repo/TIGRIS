# CODEX Design System

Visual design assets for CODEX v0.5. Direction 4 (Gilded Void) is canonical.

## Directory

```
design/
  identity/
    visual-identity.md    ← colour system, typography, component specs
    codex.css             ← shared CSS variables + base styles
  mockups/
    direction4-canonical.html   ← THE reference (open in browser)
    direction1.html             ← Arcane Academic (rejected)
    direction2.html             ← Crystalline Abstract (runner-up)
    direction3.html             ← Elemental Minimal (rejected)
  cards/
    doctrine-pool.html          ← all 25 shared Doctrine cards
    founding-flame.html         ← Flame 5-card Founding deck
    founding-tide.html
    founding-iron.html
    founding-earth.html
    founding-wind.html
    founding-ash.html
  boards/
    school-boards.html          ← all 4 base school boards (full layout)
  components/
    component-spec.html         ← Forge / Sanctum / Channel / Spire tokens
  box/
    box-art.html                ← box cover concept
```

## Open files

All HTML files are standalone — open directly in any browser, no server needed.
Google Fonts CDN used for Cinzel + Space Grotesk + Space Mono.

## Design direction summary

**Gilded Void:** Warm dark backgrounds (#13100C) + gilt accents (#C9941A) +
Cinzel serif card names + Space Grotesk rule text. School colours are
saturated and warm-shifted. Geometric SVG sigils for each School.
The Crown hex glows gold on the map.

See `identity/visual-identity.md` for full colour system and component specs.
