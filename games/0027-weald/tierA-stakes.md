# Tier-A Stakes — WEALD

## Anchor

**Kramer-Kiesling** anchors **D2 Spatial-Interaction Presence** (adopted). Claim: "WEALD's shared hex map with terrain-type adjacency bonuses creates the game's primary spatial argument: which hexes are worth fighting over, which adjacency patterns maximize VP, and which positions block opponents' optimal development. The spatial structure of the board is load-bearing throughout all 5 rounds."
