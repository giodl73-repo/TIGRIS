# Agricola — Rules Summary (Review Import)

Condensed for panel review. Canonical rules in Lookout 2016 Revised Edition.

## 1. Overview

1-5 players each manage a 17th-century farm. Over 14 rounds (4p; scales with count), each family member (worker) takes one Action per round via worker placement on a central Action Board. Rounds add new Action spaces. Harvest rounds punctuate the game: players must feed all family members or gain Begging cards (−3 VP each). By game end, players should have cultivated fields, pastures with livestock, upgraded home, expanded family, and played Occupations + Minor Improvements from private hands.

Final scoring: 14+ categories, minus penalties for missing categories.

## 2. Components

- Main Action Board (+ Round cards 1-14)
- Private Farmyard boards (one per player; 3×5 grid)
- Worker (Family Member) figures (starts at 2; grows to 5)
- Resources: wood, clay, stone, reed, grain, vegetables
- Animals: sheep, wild boar, cattle
- Food tokens
- Occupation cards (~170); Minor Improvement cards (~170); Major Improvements (14 shared)
- Begging cards (negative VP)
- Harvest tiles marking harvest rounds

## 3. Setup

Each player: 2 family members on farmyard's two rooms (Wooden House). Deal 7 Occupations + 7 Minor Improvements to hand. First-player marker placed.

Action board: basic actions revealed; round-card actions added as rounds progress.

## 4. Turn structure

Round cycle (14 rounds at 4p):
1. **Replenish** — add accumulation resources to their action spaces (e.g., +1 wood on Forest)
2. **Reveal new Action** (if applicable this round)
3. **Work** — each player in turn order places one worker on an unoccupied action space. Continue until all workers placed.
4. **Home** — workers return to players' homes
5. **Harvest** (on designated rounds — 4, 7, 9, 11, 13, 14):
   - Field phase: harvest 1 grain/veg from each field
   - Feeding phase: 2 food per family member; can't feed → take Begging card(s)
   - Breeding phase: pastures/stables with 2+ animals produce 1 new animal

## 5. Actions (the menu)

Varies by round but typical:
- Take wood, clay, stone, reed (accumulation spaces)
- Plow field (add field tile to farmyard)
- Sow (plant grain/veg on field)
- Fences (build pastures)
- Bake bread (grain → food via oven)
- Family Growth (+1 family member; requires a free room)
- Renovation (Wood → Clay → Stone home)
- Play Occupation / Minor Improvement (from hand)
- Buy Major Improvement (from shared 14)
- Take sheep / boar / cattle
- Starting Player + Food bonus
- Day Laborer (flexible resource)

~15-20 actions available at peak. Scarcity bites hardest at 4p when fewer actions per player.

## 6. Engine subsystems

- **Family**: starts 2; can grow to 5. More workers = more actions per round, BUT more mouths to feed.
- **Home**: 2-room Wood House → extend to 5 rooms → renovate to Clay → Stone. Each extension requires resources + free space.
- **Fields + Sowing**: plow fields; sow grain/veg; harvest seasonally. Grain can bake into bread (food multiplier).
- **Pastures + Animals**: build fences; graze sheep/boar/cattle; animals breed each Harvest.
- **Occupations**: cards played from hand; ongoing passive abilities (variable setup).
- **Minor Improvements**: cards played from hand; one-time effects + ongoing tech.
- **Major Improvements**: shared 14-card pool; strong abilities (ovens, pottery, etc.).

## 7. Scoring

14+ categories at endgame:

| Category | Good | Bad |
|---|---|---|
| Fields | 0-5+ | -1 if 0 |
| Pastures | 1-4+ | -1 if 0 |
| Grain | 1-8+ | -1 if 0 |
| Vegetables | 1-4+ | -1 if 0 |
| Sheep | 1-8+ | -1 if 0 |
| Boar | 1-7+ | -1 if 0 |
| Cattle | 1-6+ | -1 if 0 |
| Unused farm spaces | — | -1 each |
| Home type | Wood 0 / Clay +1/room / Stone +2/room | — |
| Family members | +3 each | — |
| Fenced stables | +1 each | — |
| Bonus (Occupations + Improvements) | variable | — |
| Begging cards | — | -3 each |

**Missing-category penalty**: -1 VP each for any of {Fields, Grain, Veg, Sheep, Boar, Cattle, Pastures} = 0.

Max theoretical: ~60 VP. Typical 4p winning: 30-45.
