# Agricola — Concept (imported for review)

- **BGG:** 31260

## Anchor: Rosenberg on C3 Scarcity Bite

Rosenberg's #1 preferred axis. Adopted v2.7. C3's canonical case: Agricola's harvest feeding phase — periodic obligation that converts abundance to shortage, starvation is real on first plays. Anchor-10 definition literally cites this pattern.

## Why Agricola for game #14

1. **Completes 8/8 designer-on-own tally.** Rosenberg anchors his own canonical game. Pattern pending since game #1. Pattern complete after this review.
2. **A4 Variance Calibration second-earning candidate.** CoB first earning (v2.11); Agricola's Occupation+Improvement deal is variance-as-resource. If A4 earns → **23rd adoption**.
3. **C6 Point-Salad test.** Agricola has 14+ distinct scoring columns (fields, pastures, grain, veg, sheep, boar, cattle, unused, home type, family, improvements, occupations, bonus, minus-points for missing types). Arguably more point-salad than Castles of Burgundy. Re-earns adopted.
4. **Re-validates adopted axes** in Rosenberg's canonical territory: C3, C4, D4, B3, A2, C1.
