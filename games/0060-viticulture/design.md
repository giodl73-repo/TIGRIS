# Viticulture: Essential Edition — Design (Review Mode: Rules Summary)

## Published rules summary

**Players:** 1–6. **Length:** 60–90 min. **Weight:** BGG 2.9.

### Seasonal structure

**Summer:** Players place workers on Summer action spaces: plant vine cards, build structures, give tours, buy and sell grapes.

**Winter:** Players place workers on Winter action spaces: harvest grapes (flip planted vines to crush pads), make wine (convert crush pad grapes to wine tokens in cellar), fill orders (match wine to order cards), draw visitor cards.

### Wine production chain (B1 mechanism)

**Vine card → Plant (Summer) → Harvest (Winter) → Crush pad grapes → Make Wine → Wine cellar aging → Fill Order (VP).** This is the core 5-stage pipeline: vine card acquisition → planting → harvesting → crushing → wine-making → order-filling.

### Worker placement + Grande Worker

Standard worker placement: first worker to a space claims it (some spaces allow multiple workers). Grande Worker can be placed even when all spaces are filled. Players start with 2 workers; build structures to gain more.

### Scoring

VP earned from filled orders (2–6 VP each), end-game structure bonuses, and residual income. First player to 20 VP triggers final round.

### Visitor cards

Summer and Winter visitor cards add variable actions (build faster, gain extra resources, etc.). Draw 2 cards per season (some structures grant more). Visitor cards provide the game's asymmetric options.

### B2 mechanism (catastrophe pressure)

Winter event cards (from the EE expansion) add seasonal catastrophe events (frost, drought, pest) that damage crops unless players prepare. This is a mild B2 contribution.

### D4 mechanism

Cellar tiers (aged wine types) create mid-game lock-in: wine placed in a higher-tier cellar ages over multiple winters before it can be sold at premium. Committing grapes to a high-tier wine is a D4 decision — it locks that resource for 2-3 turns.
