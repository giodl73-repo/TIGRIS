# Viticulture: Essential Edition — Concept (Review Mode)

## Published game summary

Viticulture: Essential Edition is a 1–6 player worker-placement wine-production Euro (60–90 min, weight 2.9). Players build wineries across two seasons (Summer: grow vines; Winter: harvest and make wine). The wine-making chain (vine card → harvest → crush → ferment → bottle → fill orders) is a multi-stage production pipeline. Visitor cards add seasonal variety.

**BGG rank:** Top 50 (BGG ID 183394). Designer: Jamey Stegmaier, Alan Tusler. Publisher: Stonemaier Games.
