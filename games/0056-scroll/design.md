# SCROLL — Design

## Overview

SCROLL is a 1–4 player simultaneous roll-and-write. Players fill a 32-cell personal manuscript scroll over 6 rounds. A shared pool of 4–8 symbol dice is rolled simultaneously; players write one symbol per cell in chosen positions. Symbols cascade: a symbol in Row 1 modifies what the same-column cell in Row 4 scores.

**Weight:** 2.5. **Length:** 30–45 min. **Players:** 1–4.

---

## Components

- 4 Scroll sheets (4 rows × 8 columns; each player keeps own sheet)
- 12 Symbol dice (3 each of 4 types: Quill/Seal/Ink/Reed)
- 1 Die bank card (shows available dice)
- VP track

---

## Turn structure (per round)

1. **Die pool construction (optional):** The active player may add 1 die from the bank to the current pool (cumulative across rounds; starts with 4 dice; ends with up to 10 dice by Round 6).
2. **Roll:** All players roll the current die pool simultaneously.
3. **Write:** Each player selects ONE symbol from the roll result and writes it in any empty cell on their scroll. Each symbol must be written in the row appropriate to the game-round (Rounds 1–2 = Row 1 priority; Rounds 3–4 = Row 2 priority; etc.; but players may write in any row).
4. **Cascade trigger (if applicable):** If a player's Row 1 is fully complete (8 cells), cascade effects activate for that player's Row 4.

---

## Cascade rule

When Row 1 is complete: each Quill in Row 1 grants +1 to the VP value of the same-column cell in Row 4. Each Seal in Row 1 grants a Column Bonus to Row 4 (entire column scores at ×2 if matching symbol in Row 4). Each Ink in Row 1 unlocks a new Row 4 cell type (hidden until Ink placed).

**Cascade depth:** Die Symbol (A4) → Row 1 Placement → Cascade Modifier → Row 4 Output → VP. This is depth-4 (B3 earn mechanism).

---

## Scoring

- Completed rows score progressively: Row 1 = 1 VP/cell, Row 2 = 2 VP/cell, Row 3 = 3 VP/cell, Row 4 = 4 VP/cell (modified by cascades).
- Symbol bonuses: complete set of all 4 symbols in any row = +5 VP row bonus.
- Full scroll bonus (all 32 cells filled): +10 VP.
