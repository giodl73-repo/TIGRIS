# Tikal — Rules Summary (Review Import)

Condensed summary for panel review. Canonical rules in Ravensburger / Rio Grande 1999 rulebook.

## 1. Overview

2-4 players race archaeological expeditions through a Mayan jungle (built from hex tiles) to uncover temples, excavate pyramids upward (levels 1–10), and collect treasures. Scoring triggers at volcano-tile draws; endgame when the tile stack is exhausted. Highest total VP wins.

**Object:** accumulate the most VP via controlled temples (level-based), matched treasure sets, and pyramid-level control, summed across 4 scoring rounds + endgame.

## 2. Components

- Hex tiles: jungle (movement), temple (excavatable pyramids, levels 1–10), treasure (goods tokens in pairs/triples), volcano (scoring-round trigger)
- Tile stack (shuffled face-down; draw 1 per turn)
- Expedition pieces per player: 1 leader, 10 workers, camp markers
- Temple level tokens (stackable, 1–10)
- Treasure tokens (6 types, multiple each)
- VP track
- Auction chits (variant)

## 3. Setup

Starting board: 3 hex tiles placed at the edge. Each player starts with leader + 3 workers at base camp. First player randomly determined. Tile stack shuffled with 4 volcano tiles inserted at pre-declared intervals.

## 4. Turn structure

Each turn, the active player:

1. **Draw 1 tile** from the stack. Place it adjacent to an existing tile (expands the jungle). If the drawn tile is a **volcano**, trigger a scoring round first, then place it (treating it as blocked terrain).
2. **Spend 10 Action Points (AP)** from the menu below. Any unused AP is lost.

### Action menu (10 AP per turn)

| Action | Cost | Effect |
|---|---|---|
| Move a worker (across clear hex) | 1 AP | Move 1 hex (jungle costs more — up to 3 AP depending on tile) |
| Uncover temple | varies (2–8 AP by tile) | Reveal hidden temple level (draw chit from tile) |
| Excavate temple level (+1 level up to 10) | 2 AP / level | Raise temple; each level = +1 VP at scoring |
| Place camp marker | 5 AP | Claim a hex as campable ground |
| Place new worker (from supply onto camp) | 1 AP | Bring reserve onto board |
| Collect treasure | 3 AP | Pick up treasure token from hex |
| Trade treasure (player-to-player) | 3 AP | Exchange matched treasures |
| Guard temple (during scoring) | varies | Out-count opponents at that temple |

Variable-cost menu with ~8 actions. The anchor-10 descriptor for C7 names this directly.

## 5. Key subsystems

### 5.1 Jungle expansion (spatial)

Every tile draw grows the map. Movement costs rise on jungle hexes. Placement is strategic — where you place affects your own access and opponents' access. Hexes interlock; adjacency dictates who can reach temples/treasures.

### 5.2 Pyramids (spatial-vertical)

Temples start at level 1; players spend AP to excavate upward (levels 2, 3, 4 ... up to 10). At each scoring round, whichever player has the most workers adjacent to a temple scores its full level value. Ties split by who placed first, or zero-out depending on variant.

### 5.3 Treasures

Treasure tokens come in 6 types. Scoring scales: 1 matched pair = 1 VP, 2 = 3 VP, 3 = 6 VP (triangle-number progression). Incentivizes breadth + trading.

### 5.4 Camp markers

Placing a camp lets a player claim a hex as a "supply base." Workers can deploy from camps instead of base-camp (reducing movement cost). Camp placement is strategic — block opponents, extend your reach.

### 5.5 Scoring rounds (triggered by volcano tiles)

When a volcano tile is drawn: pause the turn; all players declare temple guards (workers adjacent); tally VP from temples + treasures held + pyramids. 4 scoring rounds total in base game (pre-placed volcano intervals). Final scoring at endgame.

## 6. Scoring / end condition

### End condition

Tile stack exhausted → final scoring round → game ends.

### Scoring (each round)

For each player, sum:
- **Temple VP**: for each temple where you have the most workers adjacent, level value (1–10 VP)
- **Treasure VP**: triangle-number pairs (1, 3, 6 for 1, 2, 3 pairs of a matched type)
- **Pyramid bonus**: top-level control per pyramid
- Camp bonus (minor, 1 VP per camp)

Accumulated across 4 rounds + endgame.

## 7. Auction variant

Alternate rule: instead of seating order, players auction turn order each round. Adds Knizia-style auction tension. Base game uses seated order; variant is competitive-tournament default.
