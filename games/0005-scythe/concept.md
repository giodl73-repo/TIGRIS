# Scythe — Concept (imported for review)

## Premise

Review of *Scythe* (Jamey Stegmaier, 2016). First game reviewed under the v2.4 roster (8 personas, Vaccarino newly seated).

- **Players:** 1-5 (solo mode extensive; Automa AI).
- **Length:** 90-120 min (standard), ~2h first play.
- **Weight:** ~3.4/5. Mid-heavy Euro.
- **Setting:** alternate 1920s Eastern Europe; diesel-punk mechs + agrarian economy.
- **Mechanics:** asymmetric factions (5), asymmetric player mats (5), hex-territory board, action-mat turn selection, end-game multi-factor multiplier scoring.

## Anchor persona + axis

- **Anchor: Stegmaier on D3 Count-Robustness.** Parallels Knizia-on-T&E (designer-on-own-game; canonical calibration test). Scythe's 1-7p coverage (base + expansion) is the strongest count-robustness case in the Euro canon.
- Expected: D3 earns strongly; Stegmaier's first anchor success confirms the axis is well-calibrated.

## Why Scythe for game #5

1. **First game under v2.4 (Vaccarino seated).** Tests whether the 8-persona roster produces richer arguments than 7. Expected: A4/A5/B3 finally drafted; pool fully saturated.
2. **I-parliament-02 cluster test.** Scythe's end-game 6-factor multiplier (stars × territory × resources × popularity × buildings × mechs) is a direct test of the Scoring Multiplier Dependency axis candidate. If the panel notes it as a missing axis, I-parliament-02 cluster strengthens.
3. **Queued-for-adoption second-earning triggers.** A7 Emergence (Lacerda first earned in Dominion), D1 Family-to-Expert (K-K first earned in Dominion), C4 Engine-Garden (reversed in Dominion) all likely earn again — potential 3 more adoptions.
4. **v2.3 GATE check first real application.** Stegmaier-anchor on D3; D3's adjacency partner D1 drafted by K-K (cross-player). GATE passes naturally.

## Expected review outcome

- Earned: D3 (anchor), D1, D2, C4, A7, B1, B5, possibly A3
- Retire-explicit: C2 (no min-score), B2 (no catastrophes), C5 (weak anti-catch-up), A5 (Scythe has significant downtime at 3-4p)
- New-axis candidate: **Scoring Multiplier Dependency** (I-parliament-02 second instance expected)

## Non-goals

- Not testing expansion factions (base game only).
- Not resolving "solo Automa counts as count-robustness" debate — focus on multiplayer.
