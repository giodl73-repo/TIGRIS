# T&E — Concept (imported for review)

## Premise (review note)

This is not an original TIGRIS design. It is a **review** of *Tigris & Euphrates* (Reiner Knizia, 1997), the game TIGRIS is named after. The CONCEPT stage is imported — the game already exists; TIGRIS's job is to evaluate whether the v2.1 rubric can recognize what makes T&E a masterwork.

Game being reviewed:
- **Tigris & Euphrates** — Reiner Knizia, Hans im Glück (1997), Fantasy Flight Games (2015 reprint).
- 2–4 players (prefers 3–4); 90 min; weight ~3.5/5.
- Civilization tile-laying with catastrophe events and minimum-score victory.

## Why T&E is anchor game #2

1. **Calibration against a known masterwork.** The v1.0 spec §10 named T&E as the original anchor; Parliament displaced it to prove the v2.0 bet. Now that Parliament's run surfaced real evidence, T&E tests whether the rubric recognizes ambitions Parliament doesn't possess (e.g., deeper system gearing, larger emergent map arc, catastrophe-as-primary-loop).
2. **Anchor-axis continuity.** Knizia anchored Parliament on C2 Minimum-Score Shape; his own T&E is literally the canonical instance of that shape. A second earning on C2 triggers adoption (per v2.0.1 ledger queue).
3. **Queued axis triggers.** Parliament queued 3 axes for adoption (C2, B5, B4) and 2 for retirement (C4, C5). T&E evidence will trigger most of these. The factory's first adoption/retirement events happen here.

## Anchor persona + axis

- **Anchor persona: Knizia.** T&E is his signature design. If his own persona-lens cannot recognize what's best about his own game, the rubric is broken.
- **Anchor axis: C2 Minimum-Score Shape.** T&E's VP rule (final VP = MIN across 4 colors) is the canonical instance of this axis. A score below 9/10 would invalidate either the rubric or the persona.

## What T&E is expected to earn

Based on widely-held reputation:
- **Earned with 3+ marks:** Minimum-Score Shape (C2 — anchor), Catastrophe Pressure (B2), Tension Budget (C1), System Gearing (B1).
- **Earned with 2 marks:** Interaction (A3), Emergence (A7), Spatial-Interaction Presence (D2).
- **Refuted:** Point-Salad Incommensurability (C6 — T&E is minimum-score, not point-salad; a direct contrast).

If the panel's actual earned/refuted pattern deviates significantly from this expectation, the rubric is either ahead of reputation or behind it — either outcome is informative.

## Artifact-as-story

The game's centerpiece is the **board + tile bag**. The empty board at game start becomes crowded Mesopotamian kingdoms by end; the tile bag ensures every game has a different tile-emergence pattern. Catastrophe tiles are the agents of reversal; treasure tiles are the agents of concentration.

This contrasts with Parliament's artifact (the Axis Pool deck, persistent across sessions). T&E's artifact is ephemeral-within-game: every game's map is a new map.

## Target review outcome

Pass all four success criteria:
- ≥ 5 earned stakes
- ≥ 2 on Band B or C
- ≥ 1 clean collision
- ≥ 1 axis promoted or retired

With the v2.1 FORCED-ENGAGEMENT fix, the ignore rate should drop to <20% (from Parliament's 48%). If it doesn't, the fix failed and v2.2 needs a stronger version.
