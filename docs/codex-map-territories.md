# CODEX Map Territory Features

Same approach as doctrine cards: generate 50 features across 5 categories,
score each on 5 criteria, select top 4 per category = 20 features placed on the map.

---

## Scoring Rubric (max 25 points)

| Criterion | 1 | 3 | 5 |
|---|---|---|---|
| **L — Legibility** | Requires rules lookup mid-game | Understood after slow read | Grasped at a glance, never forgotten |
| **S — Strategic pull** | Ignorable | Creates a real "do I race for this?" | Everyone wants it; fight is inevitable |
| **I — Interaction trigger** | Solo benefit only | Occasionally pulls players together | Naturally causes contact every game |
| **B — Balance** | Warps the game alone OR irrelevant | Useful but never dominant | Powerful only in combination |
| **X — Cross-type** | Only good on 1 territory type | Works on 2-3 types | Interesting on any territory type |

**Selection:** Top 4 per category = 20 features placed on the 48-hex map.
Tiebreaker: higher S+I (strategic pull + interaction).

## MAP LAYOUT — CODEX HEX GRID (48 territories)

```
Conceptual zones:

         [IRON — NW]          [TIDE — NE]
      Forge/Sanctum/Ember    Flow/Sanctum/Void

  [CORRIDOR NW]  [CENTRAL RING]  [CORRIDOR NE]
                 [THE CROWN]
                 [THE NEXUS]
                 [THE LIBRARY]
                 [THE VEIL'S EYE]

      [CORRIDOR SW]           [CORRIDOR SE]

         [EARTH — SW]         [FLAME — SE]
      Sanctum/Forge/Flow    Ember/Forge/Void
```

**48 total hexes:**
- 4 starting zones × 6 hexes = 24 hexes (home terrain + 1-2 neutral)
- 4 corridor zones × 3 hexes = 12 hexes (mixed terrain, first collision round 3)
- Central zone = 8 hexes (all terrain types, highest feature density)
- Buffer ring = 4 hexes (outer edge scoring positions)

**Territory distribution:**
- Forge Grounds: 11 hexes
- Sanctum Hills: 11 hexes
- Flow Channels: 10 hexes
- Ember Peaks: 9 hexes
- Void Rifts: 7 hexes (rarest — Conflict features concentrate here)

**Starting home terrain by School:**
| School | Home hex type | Starting corner |
|---|---|---|
| Iron | Forge Grounds (×3) + Sanctum Hills (×2) + Ember Peaks (×1) | Northwest |
| Tide | Flow Channels (×3) + Sanctum Hills (×2) + Void Rift (×1) | Northeast |
| Flame | Ember Peaks (×3) + Forge Grounds (×2) + Void Rift (×1) | Southeast |
| Earth | Sanctum Hills (×3) + Forge Grounds (×2) + Flow Channel (×1) | Southwest |

**Feature placement rules:**
1. No starting zone has a Conflict feature (no punishing turn-1 combat)
2. The Crown, Nexus, and Library are central — nobody starts adjacent to them
3. Gateway appears once per starting zone edge (rewards first expansion)
4. Fault Lines sit in the corridor chokepoints — collision is forced, not optional
5. Veil's Eye is central and contested — slowing doom is a shared incentive but only one player benefits

## Design notes

**The corridor system is the game's spine.** 4 corridors of 3 hexes connect corners to center. Every game, these corridors determine which Schools collide first. Fault Lines and Chokepoints placed here mean collision has immediate mechanical consequences — not just "we're near each other" but "your engine just stopped."

**The Crown is the north star.** One hex, center of the map, worth more than anything else. 5 VP + a named blocking effect. Everyone sees it from turn 1. But reaching it means crossing 2 corridors — past Fault Lines, Chokepoints, other Schools. It's always contested and it should be.

**Void Rifts are sparse and punishing.** Only 7 hexes, mostly in corridors and near Flame's corner. War Footing and Ancient Law sit here. Void is hard to generate so these hexes have a natural cost to access — but the Conflict features they carry reward aggressive Schools (Flame) who spend there anyway.
