# CODEX Veil Events

Each time the Veil advances 1 step, the active player flips the top Veil Event card and
reads it aloud. Effects apply immediately and last until the start of the next round unless
stated otherwise.

**Thematic premise:** The Veil is the membrane between stable reality and the Void. Each
step it retreats, something cracks. Events are not punishments — they are consequences of
the Schools fighting with forces too large to fully control.

## Scoring Rubric (max 25)

| Criterion | 1 | 3 | 5 |
|---|---|---|---|
| **L — Legibility** | Requires mid-game re-read | Clear after slow read | Memorised after one game |
| **S — Strategic impact** | Negligible effect on decisions | Changes 1 player's plan | Forces every player to re-evaluate |
| **I — Interaction trigger** | Isolated / solitaire effect | Touches 1 opponent | Creates a shared situation all must respond to |
| **B — Balance** | Punishes a specific School or position | Mild swing possible | Symmetric or position-independent |
| **X — Universality** | Only relevant in one game state | Sometimes irrelevant | Always lands regardless of board state |

**Selection:** Top 4 per category (5 categories × 4 = 20 total). Tiebreaker: prefer higher I + X scores.
