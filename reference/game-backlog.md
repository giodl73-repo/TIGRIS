## Priority ranking logic

Factors for ordering:
1. **Designer-on-own-game instances** — each canonical persona should eventually review their own signature game. Calibration benefit per first-instance.
2. **Axis adoption triggers** — games that complete 2nd-earning for queued-for-adoption axes.
3. **Protocol amendment candidates** — games that surface new cluster evidence.
4. **Genre breadth** — cooperative, abstract, dice, real-time, etc. — expand TIGRIS's measurement range.
5. **BGG rank / community recognition** — classics provide strong calibration.
